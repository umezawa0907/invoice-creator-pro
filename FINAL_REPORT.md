## 🚀 **使い方ガイド**

### **初回セットアップ（5分）**

```bash
# 1. プロジェクトを展開
tar -xzf invoice-creator-pro-v3.tar.gz
cd invoice-creator-pro

# 2. 依存関係をインストール
npm install

# 3. 開発サーバー起動
npm run dev

# 4. ブラウザで開く
# http://localhost:3000
```

### **プロフィール登録**

```
1. ホームページで「設定を始める」をクリック
2. プロフィール作成フォームに入力:
   - 基本情報（氏名、住所、連絡先）
   - 銀行口座情報
   - 適格請求書番号（任意）
   - 源泉徴収計算方法のデフォルト
3. 保存ボタンをクリック
4. 完了！
```

### **請求書作成からPDF生成まで**

```
1. 「請求書を作成する」をクリック

2. Step 1: 発行者情報確認
   - プロフィールが自動選択される
   - 「次へ」をクリック

3. Step 2: 請求内容入力
   - クライアント名を入力
   - 請求項目を追加
     * 作業内容: 例「Webデザイン」
     * 数量: 1
     * 単価: 50,000円
   - 源泉徴収の有無を選択
   - リアルタイムで金額が計算される
   - 「確認へ」をクリック

4. Step 3: 確認
   - 請求内容と金額を確認
   - 「請求書を生成」をクリック

5. PDFプレビュー
   - モーダルが開く
   - 「PDFを生成」をクリック
   - プレビューで内容確認
   - 「ダウンロード」でPDF保存

6. 完了！
```
